Quarkus's Colaborative Community Forum

The core foundation of every good open source project is open discussion and collaboration. Quarkus is committed to this belief. The current social media landscape has splintered that communication across many platforms and services. This can pose a problem for our community as it becomes increasingly difficult to keep up with the conversations. In order to help the community access all of this great information, we've enabled GitHub Discussions. This gives us a focal point for the community to ask and answer questions regarding the project. This means all of that great knowledge can be shared and is discoverable. Additionally, other conversations about the Quarkus Community can be had there as well. Come collaborate with us.

Go to the Quarkus GitHub Discussions

Quarkus Chat Options
